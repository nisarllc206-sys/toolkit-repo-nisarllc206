## 📖 Description

**AI PDF Super Toolkit** is a comprehensive PDF and file utility app built by **Nexus Ultra Platforms**. It combines 100+ tools for PDF management, AI-powered analysis, file conversion, media processing, QR code generation, and more — all in a single app available on Android and as a Web PWA.

The app is powered by Claude Sonnet 4 with a GPT-4o fallback, offering intelligent, self-improving AI features that get smarter over time.

---

## ✨ Features

### PDF Operations
- **Merge PDFs** — Combine unlimited PDFs in any order with drag-and-drop reordering
- **Compress PDF** — Smart compression preserves quality while reducing file size up to 80%
- **Encrypt PDF** — AES-256 password protection for sensitive documents
- **e-Signature** — Draw, type or upload a signature to legally bind documents instantly

### AI-Powered Tools
- **AI PDF Chat** — Chat with any PDF using Claude AI — Q&A, summaries, data extraction
- **OCR Extract** — High-accuracy OCR enhanced by Claude AI — 50+ languages supported
- **AI Summarizer** — 4 summary styles (bullet, executive, simple, detailed) in 40+ languages
- **PDF Analyzer** — Deep AI analysis — key points, topics, sentiment, readability score

### Conversion & Utilities
- **PDF → Word** — Layout-perfect conversion to `.docx` preserving fonts and formatting
- **Image Convert** — Convert between JPG, PNG, WebP, HEIC with batch processing support
- **QR Generator** — Generate custom QR codes with logo, colors, and tracking analytics
- **ZIP / Unzip** — Compress and extract ZIP, RAR, 7z files with batch operations

### Platform
- Native **Material 3 Android** app
- Installable **Web PWA** — works offline
- Seamless file sync across both platforms

## 🚀 Setup & Installation

### Prerequisites
- Node.js 18+
- npm or yarn

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/nisarllc206-sys/Ai-pdf-super-toolkit-.git
cd Ai-pdf-super-toolkit-

# 2. Install dependencies
npm install

# 3. Copy the environment file and fill in your values
cp .env.example .env

# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
